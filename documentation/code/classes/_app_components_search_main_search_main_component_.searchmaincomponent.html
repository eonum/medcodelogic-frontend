Class SearchMainComponent

Container for the SearchFormComponent and SearchResultsComponent. The component is assigned to the route <catalog>/<version>/ and takes an optional query parameter.

The catalog is resolved by the CatalogResolver and then passed as input to the SearchFormComponent. Each time the query or catalog changes, the searchResults for the SearchResultsComponent are updated accordingly.
