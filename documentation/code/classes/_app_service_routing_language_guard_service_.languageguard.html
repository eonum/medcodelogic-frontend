Class LanguageGuard

This is an authentication guard that grants always access, but sets the translation according to the language parameter in the path.

@see AppRoutingModule, https://angular.io/docs/ts/latest/guide/router.html#guard-admin-feature

canActivate

  • canActivate(route: ActivatedRouteSnapshot): boolean
    • Defined in app/service/routing/language-guard.service.ts:35

    This function is a hook for the authentication, i.e. it is called always before the Route gets activated.

    In this case it applies the translation and grants access if an existing language is present in the root. Otherwise it redirects to the browser (or default) language.

    Parameters

    • route: ActivatedRouteSnapshot

      contains route params up to the point where this guard gets activated.

    Returns boolean

    • True, if a valid language parameter is in the route.
