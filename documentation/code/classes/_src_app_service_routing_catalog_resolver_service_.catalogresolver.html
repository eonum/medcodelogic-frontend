Class CatalogResolver

This service acts as resolver for a path that contains a catalog and version parameter. It gives the MainComponent access to the Catalog that corresponds to the route, and saves it for global usage.

see

AppRoutingModule, https://angular.io/docs/ts/latest/guide/router.html#resolve-guard
