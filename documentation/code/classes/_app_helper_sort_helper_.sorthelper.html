compareAsNumberWithLeadingLetter

  • compareAsNumberWithLeadingLetter(a: string, b: string): number
    • Defined in app/helper/sort.helper.ts:76

    Compares two literal with a number preceeded by letters. First, the arguments are compared lexicographically by their leading characters. Second, they're compared by their ending numbers. Returns a negative number, if the first argument is smaller than the second argument, a positive number if the first argument is larger than the second argument and 0 if they're equal.

    Examples: 'ab9', 'ab10' -> -1 'z1', 'f1' -> -1 'zz10', 'ab10' -> 1

    Parameters

    • a: string

      the first argument to compare

    • b: string

      the second argument to compare

    Returns number
